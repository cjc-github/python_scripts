使用方法：
1.运行.py程序
2.指定需要隐藏的窗口进程
3.按快捷键ctrl+z隐藏窗口，按ctrl+x恢复窗口
4.最小化本软件
点击隐藏后窗口不会在任务栏显示，点击最小化后本软件也会在任务栏消失
需要修改快捷键的可以自己在代码里改
没有库的自己根据提示安装
必须有ico图标文件才能运行
程序最小化后可在托盘右键呼出菜单
